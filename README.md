# Capital-Asset-Pricing-Model-CAPM-Analysis

This project implements the Capital Asset Pricing Model (CAPM) to estimate the expected return of individual stocks and a portfolio based on their systematic risk (Beta) relative to the market.

The analysis is performed using historical stock data and the S&P 500 index as the market benchmark.

🎯 Project Objective

Calculate daily stock returns

Estimate Beta and Alpha using linear regression

Compute expected return using CAPM

Evaluate portfolio expected return

🛠️ Tech Stack

Python

Pandas & NumPy

Matplotlib & Seaborn

Plotly

SciPy

📊 Methodology
1️⃣ Data Preparation

Loaded historical stock price dataset

Sorted data by date

Calculated daily percentage returns


📈 Outputs

Beta and Alpha for each stock

Scatter plots with regression line

Expected return of each stock

Expected return of portfolio

💡 Key Financial Concepts Used

Systematic Risk (Beta)

Market Risk Premium

Risk-Return Tradeoff

Portfolio Expected Return
